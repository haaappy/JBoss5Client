JBoss5Client用户手册

前言

本文为JBoss5Client用户手册,为使用该工具的用户提供帮助。

环境配置

  • OS: Ubuntu 12.04
  • JDK: Java Version “1.6.0 24”
  • IDE: Eclipse IDE for Java EE Developers
  • AS: JBoss AS 5.1.0

使用

运行代码，启动JBoss5的客户端图形界面。

输入参数

  • JBoss Home: JBoss应用服务器的主目录
  • Server Name: 部署服务器名称，默认为default
  • File Name: 需要执行操作的文件路径，可以通过按钮选择文件

功能按钮

  • 启动：启动选择的JBoss应用服务器
  • 停止：停止选择的JBoss应用服务器
  • 选择文件：文件路径选择
  • 部署文件：在应用服务器的部署目录中，复制所选文件，进行部署
  • 更新文件：在应用服务器的部署目录中，更新所选文件，进行替换
  • 卸载文件：在应用服务器的部署目录中，删除所选文件，进行卸载

控制台信息

将原来命令行界面的所有输出信息在图形界面上显示出来。
